```
Phase 1: 项目初始化     → 创建目录结构
Phase 2: 题目分析       → 提取PDF，分析题型
Phase 3: 数据探索       → EDA分析（如有数据）
Phase 4: 模型选择       → 推荐并选择模型
Phase 5: 大纲生成       → 生成论文大纲

```

## 详细步骤

### Phase 1: 项目初始化

**命令**:
```bash
python scripts/init_project.py --problem C --year 2026 --team "DemoTeam" --full-paper
```

**生成文件**:
- `MCM_2026_C_DemoTeam/paper/` - LaTeX论文
- `OVERLEAF_GUIDE.md` - 上传指南
- `paper_progress.md` - 进度追踪

### Phase 2-8: AI自动执行

用户只需在对话中确认每个阶段的分析结果，AI会自动：
1. 提取并分析题目
2. 探索数据特征
3. 推荐合适模型
4. 生成论文大纲
5. 撰写各章节内容
6. 组装LaTeX项目
